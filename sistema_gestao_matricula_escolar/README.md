# Sistema Gestão Matrícula Escolar - Kit gerado

Pasta contendo backend (Spring Boot), frontend (React + Vite) e scripts SQL.
Siga as instruções em backend/README.md para rodar o backend.

Observações:
- Ajuste senhas e variáveis em backend/src/main/resources/application.properties
- Rode o script SQL `database/sistema_escolar_basico.sql` no MySQL antes de executar o backend.
- Rode MongoDB localmente (ou via Docker) para logs.
